## 功能

- 更好的过滤器，日期范围，数量范围等。

- 基于Bootstrap3，支持在多种屏幕上无缝浏览，并支持Bootstrap主题模板

- 内置丰富的插件功能。包括数据导出、书签、图表、数据添加向导及图片相册等多种扩展功能

- 插件开发简单，安装方便。
